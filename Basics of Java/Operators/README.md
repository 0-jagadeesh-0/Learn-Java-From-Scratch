# Java Operators

Java Operators are classified below

- **Arithmetic Operators**
- **Unary Operators**
- **Assignment Operators**
- **Relational Operators**
- **Logical Operators**
- **Ternary Operator**
- **Bitwise Operators**
- **Shift Operators**

### Arithmetic Operators
These operators are used to peform simple arithmetic operations.
- **Multiplication** - <strong>*</strong>
- **Division** - **/**
- **Modulo**  - **%**
- **Addition** - **+**
- **Subtraction** - **-**
